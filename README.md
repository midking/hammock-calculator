# 해먹 각도 계산기

Plagenhoef 인체 무게 비율과 비대칭 처짐을 반영한 해먹 설치 각도 계산기.

## 주요 기능

- **각도 계산**: 머리쪽·다리쪽 앵커에서 릿지라인과 해먹 사이의 각도
- **무게중심 표시**: 인체 무게중심(배꼽 기준, 머리에서 44%) 위치 시각화
- **장력 표시**: 지렛대 원리 기반 각 앵커의 장력 비율
- **기울기 지원**: 다리쪽 앵커 높이 조절 시 기울어진 설치 반영
- **릿지라인 자동 계산**: 전체 줄 길이의 83% 권장값 자동 적용
- **SVG 다이어그램**: 설정값에 따른 실시간 시각화
- **다크모드**: `prefers-color-scheme` 자동 대응

## 입력값

| 항목 | 범위 | 설명 |
|------|------|------|
| 해먹 길이 | 200~400 cm | 해먹 본체 길이 |
| 연장선 머리쪽 | 0~80 cm | 머리 앵커와 해먹 사이 줄 |
| 연장선 다리쪽 | 0~80 cm | 다리 앵커와 해먹 사이 줄 |
| 릿지라인 | 150~420 cm | 두 앵커 사이 직선 거리 (수평) |
| 다리쪽 앵커 높이 | 0~80 cm | 다리쪽 나무가 머리쪽보다 높은 정도 |

## 계산 원리

- **무게중심 위치**: Plagenhoef(1971) 인체 세그먼트 데이터 기반, 머리에서 약 44% (배꼽 부근)
- **장력 비율**: 지렛대 원리 — 무게중심에서 가까운 앵커가 더 큰 장력을 받음
- **각도**: 릿지라인(현)을 기준으로 해먹 처짐(sag) 기하학으로 계산
- **권장 설치 각도**: 평균 20°~35° (이상적 범위)
